# pp-app — Online Courses Catalog

Pixel-perfect верстка каталога онлайн-курсов по макету Figma.

## Технологии

- HTML + BEM
- SCSS → CSS (Sass)
- Vanilla JS
- Playwright (скриншотные тесты)

## Установка

```bash
npm install
npx playwright install chromium
```

## Запуск

```bash
npm run start    # локальный сервер на http://127.0.0.1:8080
```

## Скриншотные тесты

Тесты сравнивают мой вариант верстки с эталоном из Figma (`Test Task.png`) на уровне пикселей и сохраняют три изображения для визуального сравнения.

### Запуск

```bash
npm run screenshots
```

### Результаты

После запуска в папке `test-results/` появятся три файла:

| Файл | Описание |
|------|----------|
| `эталон-1920x1893.png` | Эталонный макет из Figma |
| `мой-вариант-1920x1893.png` | Скриншот моей верстки |
| `диф-1920x1893.png` | Разница между ними (красные пиксели = расхождения) |

В консоли выводится процент совпадения:

```
Pixel match: 98.45%  (diff pixels: 1823 / 3624960)
```

### Как читать диф

Откройте `test-results/диф-1920x1893.png` — красные области показывают, где верстка отличается от макета. Чем меньше красного, тем точнее результат.
